# Dark Souls WebTools
[![Django CI](https://github.com/AlessVolpe/DarkSouls_WebTools/actions/workflows/django.yml/badge.svg)](https://github.com/AlessVolpe/DarkSouls_WebTools/actions/workflows/django.yml)

Dark Souls WebTools is a work in progress project, born from my passion for those games. My vision is to set up a simple application to help people build their character and optimize their exploration in those wonderful games. While the games are really popular
and old enough to be datamined to their bones, I founded it difficult find any tool online different from a build planner. Hopefully my vision is going to satisfy many.
